# 欢迎使用RevivedLocyItem!  
> 你现在在重生版LocyItem的wiki界面

你可以通过wiki来获取该插件的使用方法！  
本插件需要PlaceHolderAPI插件支持运行    
如果乱码，可以使用notepad++自行转码……  
  
基础:  
指令列表:  
/rli version —— 查看插件主版本  
/rli reload —— 重载插件  
/rli i [物品名] —— 获取一个RPG物品  
/rli p [粒子效果组名] —— 在脚底下播放一个粒子效果组  

权限:  
除了指令/rli version 以外，都需要权限 RevivedLocyItem.admin 来使用……
